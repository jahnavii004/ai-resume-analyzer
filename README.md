# 🚀 AI-Powered Resume Analyzer

An ATS-style Resume Evaluation System that analyzes resumes against target job roles and provides structured feedback.

## 🔍 Features

- 📄 PDF Resume Upload
- 🎯 Role-Based Skill Matching
- 📊 Skills Match Percentage
- 🤖 ATS Compatibility Score
- 💡 Smart Improvement Suggestions
- 📈 Visual Skill Breakdown Chart
- 🌐 REST API Backend

## 🛠 Tech Stack

Frontend:
- HTML
- Tailwind CSS
- JavaScript
- Chart.js
- PDF.js

Backend:
- Node.js
- Express.js
- Multer
- PDF Parsing

## 🎯 Supported Roles

- Software Engineer (SDE)
- Data Analyst
- Backend Developer
- Frontend Developer
- Full Stack Developer
- DevOps Engineer

## 🚀 How To Run Locally

1. Clone repository  
2. Install backend dependencies: npm install
3. Create `.env` file  
4. Run backend: npm run dev
5. Open frontend using Live Server
